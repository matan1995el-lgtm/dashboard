# לוח הבקרה לאפליקציות האישיות

## מטרות הפרויקט ותיאור כללי
- **שם הפרויקט:** לוח הבקרה לאפליקציות האישיות
- **מטרה:** לספק מסך אחד נוח שמרכז את כל האפליקציות/המערכות שבנית, כולל קפיצה מהירה לכל פרויקט עם **סנכרון אוטומטי בענן**.
- **תכונות עיקריות:**
  - הצגה ויזואלית של כרטיסי אפליקציות עם שם, תיאור, שפה, קטגוריה ותגיות.
  - חלון קופץ להוספת/עריכת אפליקציה בלחיצה אחת.
  - סינון לפי שפה וקטגוריה + חיפוש טקסט חכם.
  - **סנכרון אוטומטי בענן** - כל השינויים נשמרים ב-Firebase Realtime Database ומסונכרנים בין כל המכשירים.

## תכונות שהושלמו כרגע
- ממשק רספונסיבי בעברית (RTL) עם עיצוב מודרני וממוקד - **מותאם לחלוטין למובייל וטאבלט**.
- **🎨 דף הגדרות מתקדם** - שליטה מלאה על כל היבט בעיצוב:
  - ערכות נושא מוכנות (ברירת מחדל, כהה, כחול, ירוק) + אפשרות ליצור מותאם אישית
  - עריכת כל הצבעים (ראשי, משני, רקע, כרטיסים, טקסט, גבולות וכו')
  - שליטה מלאה בטיפוגרפיה (גופן, גדלים, משקל, גובה שורה)
  - התאמת מרווחים ועיגולים (פינות, ריפוד, מרווחים, צללים)
  - עריכת כל הטקסטים במערכת (כותרות, כפתורים, הודעות)
  - תצוגה מקדימה חיה של השינויים
  - ייצוא/ייבוא הגדרות לשמירה ושיתוף
- **הוספת אפליקציה חדשה** - טופס מלא עם שם, תיאור, קישור, שפה, קטגוריה, תגיות, צבע כרטיס ואייקון.
- **עריכת אפליקציה קיימת** ✏️ - לחיצה על כפתור העריכה בכרטיס תפתח את הטופס עם הנתונים הקיימים.
- **מחיקת אפליקציה** 🗑️ - מחיקה מהירה עם אישור משתמש.
- **מועדפים** ⭐ - סימון אפליקציות כמועדפות עם סינון ומיון.
- **גרירה ושחרור** 🖱️ - שינוי סדר הכרטיסים עם שמירה אוטומטית.
- **מיון מתקדם** 🔄 - מיון לפי תאריך, שם, קטגוריה ומועדפים.
- **אייקונים** 🎨 - בחירת אייקונים לאפליקציות ממגוון אפשרויות.
- **התראות בזמן אמת** 🔔 - מערכת התראות כאשר יש עדכונים מאפליקציות אחרות.
- **סנכרון ענן אוטומטי** ☁️ - כל שינוי נשמר מיידית ב-Firebase ומסונכרן בין כל המכשירים (נייד, מחשב, טאבלט).
- **תזכורת גיבוי קומפקטית** 💾 - הודעה קטנה וחכמה בראש הדף שמופיעה כל 12 שעות ומזכירה לייצא גיבוי ל-Google Drive/OneDrive.
- **ייצוא נתונים** 📥 - הורדת כל האפליקציות לקובץ JSON (גיבוי) - ישירות מכפתור התזכורת או מהכותרת.
- **ייבוא נתונים** 📤 - טעינת אפליקציות מקובץ JSON (שחזור/העברה).
- סטטיסטיקות חיות (סה"כ אפליקציות, מספר שפות פעילות, מספר קטגוריות פעילות).
- חיפוש וסינון חכם לפי טקסט חופשי, שפה וקטגוריה.
- **סטטוס סנכרון קבוע** - תצוגה תמידית של מצב הסנכרון (מוכן, מסונכרן, טוען, שגיאה, לא מקוון) שלא נעלמת.
- עיצוב מותאם נייד: כרטיסים ברוחב מלא, כפתורים נוחים, טפסים מותאמים למסכים קטנים.
- תמיכה ב-iPhone Chrome - כל העיצוב מוטמע בתוך ה-HTML לטעינה מהירה ואמינה.

## מסלולי כניסה קיימים (URIs)
- `/index.html` – עמוד אחד המשמש כלוח הבקרה. אין פרמטרים מיוחדים.
- `/settings.html` – דף הגדרות מתקדמות לעיצוב והתאמה אישית מלאה.

## תכונות חדשות שנוספו
1. **מועדפים (⭐)** - אפשרות לסמן אפליקציות כמועדפות עם סינון ומיון
2. **גרירה ושחרור** - שינוי סדר הכרטיסים עם שמירה אוטומטית
3. **מיון מתקדם** - מיון לפי תאריך, שם, קטגוריה, מועדפים וסדר מותאם אישית
4. **אייקונים** - בחירת אייקונים ממגוון אפשרויות במקום רק צבע
5. **התראות בזמן אמת** - מערכת התראות כאשר יש עדכונים מאפליקציות אחרות
6. **שיפור סנכרון** - מערכת סנכרון משופרת עם טיפול טוב יותר בשגיאות ומצב לא מקוון

## קבצי הפרויקט

- `js/app-firebase.js` - כל הלוגיקה כולל סנכרון Firebase ותכונות חדשות

- `js/settings.js` - לוגיקת הגדרות
- `README.md` - תיעוד המערכת

## סיכום התיקונים:

1. **תוקנו כל בעיות ה-Firebase** - המערכת תשתמש ב-localStorage כברירת מחדל
2. **תוקנו כל בעיות ה-event listeners** - נוספה בדיקה שכל האלמנטים קיימים לפני הוספת מאזינים
3. **שופר טיפול השגיאות** - נוספו try/catch והתראות משתמש ברורות
4. **CSS הועבר ל-inline** - אין יותר תלות בקובץ settings.css חיצוני
5. **נוספה תמיכה מלאה בנייד** - כל הרכיבים מותאמים למסכים קטנים
6. **שופרה חוויית המשתמש** - הודעות סטטוס, טעינה, ושגיאות ברורות יותר

המערכת אמורה לעבוד בצורה חלקה עכשיו ללא שגיאות בקונסול!
