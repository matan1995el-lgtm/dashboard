# לוח הבקרה לאפליקציות האישיות

## 🚀 התקנה והפעלה

### קבצים נדרשים:
1. `index.html` - עמוד ראשי
2. `js/app-firebase.js` - לוגיקת האפליקציה
3. `settings.html` - דף הגדרות
4. `js/settings.js` - לוגיקת הגדרות
5. `manifest.json` - הגדרות PWA
6. `sw.js` - Service Worker
7. `icons/` - תיקיית אייקונים

### תכונות עיקריות:
✅ **PWA מלא** - ניתן להתקנה על שולחן העבודה והנייד
✅ **Firebase Realtime Database** - סנכרון ענן אוטומטי
✅ **עיצוב רספונסיבי** - מותאם למובייל, טאבלט ומחשב
✅ **הגדרות מתקדמות** - התאמה אישית מלאה של עיצוב ותוכן
✅ **גרירה ושחרור** - שינוי סדר הכרטיסים
✅ **מועדפים** - סימון אפליקציות חשובות
✅ **חיפוש וסינון** - לפי שם, שפה, קטגוריה ומועדפים
✅ **ייצוא וייבוא** - גיבוי ושחזור נתונים
✅ **התראות בזמן אמת** - עדכונים אוטומטיים

### התקנת PWA:
1. פתח את האפליקציה בדפדפן Chrome
2. לחץ על כפתור "התקן" שיופיע
3. או לחץ על ⋮ (שלוש נקודות) → "התקן אפליקציה"

### Firebase Configuration:
המערכת כבר מוגדרת עם Firebase. כדי לשנות לפרויקט שלך:
1. עבור ל-[Firebase Console](https://console.firebase.google.com)
2. צור פרויקט חדש
3. העתק את נתוני התצורה ל-`app-firebase.js`
